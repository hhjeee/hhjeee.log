## 목표

동아리에서 next.js 스터디를 진행하는데, 이론 수업의 결과물로 각자의 블로그를 만들어보기로 결정하였다.<br/>
현재 기준 가장 최신 버전인 Next.js15를 사용하여 나만의 블로그를 만들어 보자 !

## 계획

### 필수 기능

- 게시글 목록 페이지(전체, 카테고리별)
- 게시글 상세 페이지

### (넣고 싶은) 부가 기능

- 목차 사이드바(TOC, Table Of Content)
- progress status bar
- 다크/라이트 모드
- 검색 엔진 최적화 (SEO)

### 기술스택

- Next.js
- Tailwind CSS
- gray-matter + next-mdx-remote(글 파싱)
  - 글을 mdx 파일로 작성해 프로젝트 내부 디렉토리에 저장
  - 구현하려는 서비스 자체가 ‘개인’ 블로그로, 여러 사용자가 글을 게시하는 방식이 아닌 혼자 글을 게시하는 방식으로 진행될거 같아 서비스 내에서 게시글을 작성하는 기능 구현이 우선이 되지 않아도 될 것 같다고 판단하였다.
