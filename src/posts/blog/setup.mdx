### 목표

나만의 개발블로그를 만들어 보자 !

### 들어가기 전에 ..

**next.js를 사용하는 이유** (꼭 사용해야 할까 ?)<br/>

1. 블로그 개발, 그리고 게시물 자체를 mdx로 저장해 사용하는 만큼 next.js의 ssr, ssg 기능을 적극 활용할 수 있을 것 같다. + seo도 ..! <br/>
2. react를 사용하고 ssr를 로우레벨에서 직접 개발해보는 것도 좋은 경험이겠지만, 아직 next.js의 고유 기능 ssr도 제대로 사용해본적이 없기에 .. next.js를 사용해보는 것으로 결정

### next.js 설치

```
npx create-next-app@latest
```

### yarn 설치

```
npm install -g yarn
yarn install
```

### font 설정

1. pretendard 다운
2. app/fonts에 PretendardVariable.woff2 파일 추가
3. layout.tsx에 아래 코드 추가

```
import localFont from "next/font/local";

const pretendard = localFont({
src: "../fonts/PretendardVariable.woff2",
display: "swap",
variable: "--font-pretendard",
});

-

<body className={pretendard.className}>
```

next/font/local을 활용하면 googlefonts가 아닌 src 내의 font 파일도 불러와 사용할 수 있음
